# EAgent
Evolutive Agent

Evolutive Agent, clearly powered by AI.

This is a private project. Built primarily in Google Antigravity, thanks to Gemini 3.1 Pro Preview & Claude Sonnet 4.6 (Thanks to these two wonderful LLMs, I’ve been able to program by talking to them).

The idea is that this Agent isn’t just useful and personal like other projects (e.g., Openclaw), but that it’s safe to install on a personal PC and powerful.

For now, I’m programming it using the Gemma 4 31B “brain” model—a wonderful LLM despite its size.

I might make a version for “the public,” but for now it’s just a private and personal project.

DETAILS:

This Agent has emotions. Well, it fakes them.
It fakes mood swings, emotional states, and has super-sharp memory skills, along with the use of embeddings like BGE-M3 and a simple file to get everything started.

Something I love is the fact that it has features like a built-in web browser and a more “detailed” web search—instead of being a standard search engine, it uses a fast LLM, Gemini 3.1 Flash Lite, which looks at 30 results from SearXNG (Local) and summarizes each one to feed them to the “master” model.
